### Estructura del test: Hooks

* before(): antes del bloque de tests
```typescript
describe('Hooks', function() {
before(function() {
  // Código
})
```
* after(): después del bloque de tests
```typescript
after(function() {
  // Código
})
```
* beforeEach(): antes de cada test
```typescript
beforeEach(function() {
  // 
})
```
* afterEach(): después de cada test
```typescript
afterEach(function() {
  // Código
})
```
