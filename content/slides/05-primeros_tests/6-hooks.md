### Hooks

before(): se ejecuta UNA VEZ antes del bloque de tests.
```typescript
before(function() { })
```

beforeEach(): se ejecuta antes de cada test.
```typescript
beforeEach(function() { })
```


after(): se ejecuta UNA VEZ después del bloque de tests.
```typescript
after(function() { })
```

afterEach(): se ejecuta después de cada test.
```typescript
afterEach(function() { })
```
